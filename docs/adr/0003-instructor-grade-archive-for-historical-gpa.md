# Instructor Grade Archive For Historical GPA

MVP-1 uses UCSD's Instructor Grade Archive as the primary Historical GPA Data source instead of the starter GPA CSV files. The archive can be queried by subject keyword and returns subject, course, year, quarter, title, instructor, GPA, and grade-bucket percentages, which is sufficient for course and instructor-course GPA display without using SET/CAPE or a personal UCSD login.
