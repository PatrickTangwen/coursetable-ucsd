## Read Order For Historical Context

this order:

1. `../../../CONTEXT.md` for shared domain vocabulary and stable product terms.
2. `../../README.md` for stable project documentation.
3. `../../adr/README.md` for durable decisions.
4. The specific archived planning doc named by the issue or ADR.
