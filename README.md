# UCSD Course Discovery and Schedule Planning

This website helps UCSD students explore course offerings, compare sections,
review historical grade context, and build schedules for supported academic
terms.

## Features

- Search courses by course code, title, instructor, subject, building,
  day/time, course number, enrollment range, and course attributes.
- Browse supported current and historical terms.
- Review course descriptions, sections, meeting times, instructors, units,
  prerequisite text, restrictions, source catalog links, and snapshot-based
  section availability.
- View historical GPA context from UCSD Instructor Grade Archive records.
- Add supported sections to a worksheet and view the result as a calendar or
  list.
- Identify schedule conflicts, hide courses from the calendar, adjust course
  colors, and switch between supported worksheet terms.
- Export a worksheet as an `.ics` calendar file or copy a shareable worksheet
  URL.
- Plan anonymously with browser-local worksheet storage.
- Sign in with a verified `@ucsd.edu` email to use account-backed saved
  worksheets and saved filters.

## Data Sources and Freshness

Course information is assembled from public UCSD sources, including:

- UCSD Schedule of Classes
- UCSD General Catalog
- UCSD Instructor Grade Archive

The website uses periodically published catalog snapshots. Enrollment,
capacity, and waitlist values reflect the snapshot publication time; they are
not live WebReg data and should not be relied on as real-time availability.
Always confirm official course and enrollment information through UCSD systems.

## Accounts and Saved Data

An account is not required for basic course planning. Anonymous worksheets are
stored in the current browser and can be shared or restored through supported
worksheet URLs.

After signing in with a verified UCSD email address, users can store
account-owned worksheets and saved filters. Account-backed data is separate
from anonymous browser storage, and signing in does not automatically import or
merge a local worksheet.

## Current Limitations

The website does not provide:

- Real-time enrollment, seat, waitlist, or demand tracking.
- SET/CAPE results or personal UCSD account scraping.
- Official UCSD enrollment actions.
- Google OAuth or direct Google Calendar write access.
- Social planning, friends, wishlists, or public worksheet permission controls.
